# Changelog

All notable changes to published packages are documented here and in per-package changelogs after the first changeset release.

## 0.1.0 — 2026-06-09

Initial public release.

### @cyanium/tokens

- AlignUI CSS variables, typography, shadows, and Tailwind v4 theme bridge

### @cyanium/ui

- shadcn/Radix primitives (Button, Input, Dialog, DataTable, and 30+ components)
- Surface tokens (`surfaceVariants`, `insetBorder`)
- shadcn registry manifest with CI drift guard

### @cyanium/kits

- Finance, landing, and AI product kits with explicit props
- `@cyanium/kits/patterns` for dashboard chrome
- `@cyanium/kits/fixtures` with `build*AppProps()` builders
- Storybook integration stories and example app
